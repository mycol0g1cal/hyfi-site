Consent-first personal finance, powered by Plaid.

What is HyFi?

HyFi is a private beta application that lets you explore your personal finances with strict control over consent and data retention. We connect to your financial accounts using Plaid so you can review balances, transactions, and spend trends with confidence that everything is auditable and reversible.

Consent-first

Every sync is tied to a Plaid Link session and logged with a consent identifier. You decide which accounts to connect, and you can revoke access at any time directly from Plaid.

Readable Receipts

HyFi keeps human-readable receipts for every run (balance summaries, ledger snapshots, drill reports) so you always know what data was accessed and why.

Data Retention

We store Plaid access tokens securely, retain financial artifacts only while you are actively using HyFi, and honour deletion requests within 30 days. Tokens are rotated or revoked on request.

Current Status

HyFi is currently in a private beta. Access is limited to internal testers and invited participants while we finish sandbox validation and Plaid production onboarding. Existing testers can continue to use the app at app.hyfi.dev.

Need Support?

If you have trouble linking an institution, or want us to revoke access to a connected account, please email support@hyfi.dev.

You can also revoke HyFi’s access yourself at any time from the Plaid dashboard: my.plaid.com.

Plaid & Security Details

  • Plaid Link is required to connect each financial institution.
  • HyFi uses Plaid’s development environment for testing and will complete Plaid’s production review before launch.
  • All traffic between HyFi and Plaid is encrypted with TLS 1.2+.
  • Financial artifacts generated by HyFi are encrypted at rest and deleted within 18 months unless you request otherwise.
